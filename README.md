# 🛡️ SafeGuard AI

Project Brief: SafeGuard AI is an AI-powered personal safety application that uses on-device AI, smartphone sensors, and IoT integration to detect potential emergencies and automatically trigger SOS alerts with location sharing.

## ✨ Features
- AI-based emergency detection
- Automatic SOS alerts
- GPS location sharing
- Edge AI for privacy
- Low-network support
- IoT integration

## 🛠️ Tech Stack
- Android Studio
- Java / Kotlin
- TensorFlow Lite
- Machine Learning
- Edge AI
- IoT
